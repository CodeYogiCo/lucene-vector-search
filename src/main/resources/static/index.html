ℹ️ How does this search work? (click to learn)

This demo searches ~1,000 products three different ways. Pick a Mode above and watch how the results change.

🧠 Vector
Finds products by meaning. A neural model (MiniLM) turns your words into a 384-number "embedding" and finds items that are conceptually close — even with no shared words. Searching "comfy ear buds" surfaces "wireless headphones".
🔤 BM25
Classic keyword search. It matches the actual words you typed and ranks by how rare and frequent those words are. Matched words get highlighted. Great for exact terms, model numbers, brands.
⚡ Hybrid
Runs both at once and blends the scores. Each result shows a badge — Vector, BM25, or Both — telling you which signal found it. Usually the best of both worlds.

The Category filter is applied inside the search (not after), so picking a category never returns items from other categories — even in semantic mode.
